# FoxPhantom Portfolio Template

A clean, modern, responsive single-page portfolio / personal site template built with HTML, CSS, and Tailwind. Perfect for developers, creators, and architects who want a sleek digital presence.

## Features
- Beautiful animated hill landscape background
- Glassmorphism project cards with horizontal scrolling
- Dark theme ready
- Fully responsive
- PWA-ready (manifest + service worker)
- Easy to customize

## Quick Start - Make Your Own Portfolio

### Option 1: Fork (Recommended for personal use)
1. Fork this repository
2. Rename the repo to your username or project name
3. Customize `index.html`, CSS files, and assets
4. Deploy for free on Vercel, Netlify, or GitHub Pages

### Option 2: Branch & Customize
1. Create a new branch: `git checkout -b my-portfolio`
2. Make your changes
3. Push and deploy

### Option 3: Contribute Back
- Found a bug? Open an **Issue**
- Have an improvement? Submit a **Pull Request**
- Want to add a new feature? Discuss it first!
